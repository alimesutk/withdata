# PROJECT NAME : withdata

### How to use postgre_compose.yml file?
  
  for compose up/down: 
  
```sh
    $ docker-compose -f postgre_compose.yml up | down
```
  for start/stop container : 

```sh
    $ docker start | stop <container id> [ <container id2> <container id3> ...]
```
